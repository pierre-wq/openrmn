How this demo works
openRMN aggregates data from 3 retail media networks (Amazon Ads, Criteo, Unlimitail) through their APIs. The data shown here is simulated but faithfully reproduces the format and structure of the real APIs. Each network attributes sales using ITS OWN methodology — that's exactly the problem openRMN exposes.
